
Небольшой Vue 3-проект с неоновой авторизацией.
Технолоджии
- Vue 3 
- Pinia 
- Vue Router
- TailwindCSS
- vue-the-mask (маска телефона)


Функции
- Авторизация по email+пароль или телефону
- Маска ввода телефона 
- Состояние авторизации сохраняется в localStorage

Запуск проекта

npm install<br>
npm run dev
